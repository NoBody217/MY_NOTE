# 4.链式前向星
- 开一个结构体存边信息，开一个数组存每一条边，开一个head记录连接着这个点的下一个点是谁,head初值赋值成-1
```cpp
typedef struct{
    int t,val,nxt;
}E;
E edge[N];
int head[N];
int cnt;

void insert(int x,int y,int l){
    edge[++cnt].t=y;
    edge[cnt].val=l;
    edge[cnt].nxt=head[x];
    head[x]=cnt;
}
```
